# Football Betting Dashboard

Fully automated dashboard using free data from football-data.co.uk.

**Features**
- Auto-updates every 2 hours (GitHub Actions)
- Poisson model for match predictions
- Value bet detector (compares model probs vs bookie odds)
- Clean web dashboard (hosted on Netlify)

## How to use
1. Data updates automatically via GitHub Actions
2. Visit the Netlify URL (deployed from this repo)
3. Look for **green rows** = value bets

Built as a simple starter using the open-source repos from @zostaff's post.
